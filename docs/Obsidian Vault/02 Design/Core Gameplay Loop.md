# Core Gameplay Loop

## The Loop

1. **Scavenge** the property for supplies (limited time)
2. **Manage Weight** — decide what to carry vs leave
3. **Return** to shelter and store excess
4. **Fortify** using right-click → place system
5. **Survive** the storm phases

## Tension Sources

- Shrinking prep timer
- Carry weight limit
- Storm intensity (affects final survival)

## Post-Jam Expansion Ideas

- Multiple properties / locations
- Survivor NPCs that need saving (extra weight cost)
- Dynamic weather affecting placement
